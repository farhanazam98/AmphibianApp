Amphibians
==================================

Amphibians app is a demo app that showcases various amphibian species. The data includes images and details about different amphibians, sourced from a REST web service.

This app demonstrates the use of Retrofit to make REST requests to the web service, kotlinx.serialization to handle the deserialization of the returned JSON to Kotlin data objects, and Coil to load images by URL.


Getting Started
---------------

1. Install Android Studio, if you don't already have it.
2. Download the sample.
3. Import the sample into Android Studio.
4. Build and run the sample.
